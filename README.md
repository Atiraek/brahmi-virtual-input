# 𑀩𑁆𑀭𑀸𑀳𑁆𑀫𑀻 Keyboard

A responsive browser-based virtual keyboard for the **Brahmi script** (𑀩𑁆𑀭𑀸𑀳𑁆𑀫𑀻, U+11000–U+1107F).

Tap keys to compose text, copy to clipboard, paste anywhere. Works on any device with a browser — no installation, no OS-level keyboard changes.

## Use

Open `index.html` in any browser — works offline. Alternatively host it anywhere (GitHub Pages, static server, etc.).

## Design

- **v1** — Direct key input. Keys grouped by linguistic category (vowels, consonant vargas, diacritics).
- **v2** (planned) — Phonetic transliteration (type "namaha" → "𑀦𑀫𑀂")

## Font Support

Brahmi glyphs require a font that covers the Unicode Brahmi block. [Noto Sans Brahmi](https://fonts.google.com/noto/specimen/Noto+Sans+Brahmi) is the best option.
